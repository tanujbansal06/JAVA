# Student Grade Manager (Java)

A simple CLI-based Java application to manage student records and marks.

## Features

* Add Student
* Store Student Marks
* Display Students

## Concepts Used

* Java OOP (Classes & Objects)
* ArrayList
* Loops
* Scanner (User Input)

## Project Structure

student-grade-manager
│
├── Main.java
├── Student.java
└── GradeManager.java

## How to Run

1. Compile the files

```
javac Main.java
Student.java
GradeManager.java
```

2. Run the program

```
java Main
```
